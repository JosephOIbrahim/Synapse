# SideFXLabs → SYNAPSE RAG Extraction

## Mission
Extract structural knowledge from `sideeffects/SideFXLabs` (BSD 2-Clause) to bolster SYNAPSE's RAG corpus, parameter discovery, and MCP tool accuracy.

## Prerequisites
- Git installed and accessible
- Python 3.10+
- Access to `G:\HOUDINI21_RAG_SYSTEM\` 
- ~500MB disk for staging

## Quick Start
```bash
python scripts/orchestrator.py
```
This runs all phases sequentially. Or run individual phases:
```bash
python scripts/orchestrator.py --phase 0   # Clone & stage
python scripts/orchestrator.py --phase 1   # Parameter schemas
python scripts/orchestrator.py --phase 2   # VEX patterns
python scripts/orchestrator.py --phase 3   # Python patterns
python scripts/orchestrator.py --phase 4   # Help intent pairs
python scripts/orchestrator.py --phase 5   # HDA conventions
python scripts/orchestrator.py --phase 6   # PDG patterns
python scripts/orchestrator.py --phase 7   # RAG merge
```
