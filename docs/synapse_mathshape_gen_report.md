## Randomization System

The `RND(base, lo, hi, seed_offset)` macro:
```vex
#define RND(base, lo, hi, s) \
    fit01(rand(rand_seed * 7.3917 + s * 1.6180339), lo, hi) * rand_amt \
    + base * (1.0 - rand_amt)
```
- At `rand_amount = 0.0` → pure canonical form (base values only)
- At `rand_amount = 1.0` → fully random within musically-coherent ranges
- At `rand_amount = 0.5` → blended: recognisable but mutated
- Each governing parameter gets a unique seed offset (multiples of golden ratio φ)

## Usage Tips

1. **Shape exploration:** Scrub `Random Seed` with `Randomize Amount = 0.5`
2. **Animation:** Drive `Twist / Phase` with `$T * 2 * $PI` for morph-over-time
3. **Tube geometry:** Connect → Sweep SOP with circle cross-section
4. **Shading:** Per-point `Cd` is always authored; use "Use Point Color" material
5. **Torus Knot best settings:** Resolution ≥ 1024, Close Curve ON
6. **Circuit traces:** Pipe into Mountain SOP → Polywire
